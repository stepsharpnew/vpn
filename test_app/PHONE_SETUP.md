# Запуск приложения на телефоне

## Вариант 1: Android эмулятор (уже установлен)

### Запуск эмулятора:
```bash
cd test_app
flutter emulators --launch Medium_Phone_API_36.1
```

### Запуск приложения на эмуляторе:
```bash
cd test_app
flutter run
```

## Вариант 2: Реальный Android телефон

### Шаг 1: Включите режим разработчика на телефоне
1. Откройте **Настройки** → **О телефоне**
2. Нажмите 7 раз на **Номер сборки**
3. Появится сообщение "Вы стали разработчиком"

### Шаг 2: Включите отладку по USB
1. Откройте **Настройки** → **Для разработчиков**
2. Включите **Отладка по USB**
3. Подключите телефон к компьютеру через USB

### Шаг 3: Проверьте подключение
```bash
cd test_app
flutter devices
```

Должен появиться ваш телефон в списке.

### Шаг 4: Запустите приложение
```bash
cd test_app
flutter run
```

## Вариант 3: iOS (только на Mac)

Если у вас Mac и iPhone:
```bash
cd test_app
flutter run -d ios
```

## Полезные команды:

- `flutter devices` - показать все доступные устройства
- `flutter run -d <device_id>` - запустить на конкретном устройстве
- `flutter run --release` - запустить в release режиме (без DEBUG)

## Устранение проблем:

Если телефон не определяется:
1. Убедитесь, что USB отладка включена
2. Разрешите отладку на телефоне (появится запрос)
3. Установите драйверы для вашего телефона (если нужно)
4. Попробуйте другой USB кабель/порт
