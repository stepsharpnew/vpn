# Быстрый запуск на телефоне/эмуляторе

## Шаг 1: Запустите эмулятор (если еще не запущен)

В терминале выполните:

```bash
cd test_app
flutter emulators --launch Medium_Phone_API_36.1
```

Подождите 1-2 минуты, пока эмулятор полностью загрузится.

## Шаг 2: Проверьте, что эмулятор готов

```bash
cd test_app
flutter devices
```

Должен появиться Android эмулятор в списке (не offline).

## Шаг 3: Запустите приложение

```bash
cd test_app
flutter run
```

Или укажите конкретное устройство:

```bash
flutter run -d emulator-5554
```

## Если эмулятор не запускается:

1. Убедитесь, что Android Studio установлен
2. Проверьте, что эмулятор создан в Android Studio
3. Попробуйте запустить эмулятор из Android Studio вручную

## Альтернатива: Запуск на реальном телефоне

1. Включите режим разработчика на телефоне
2. Включите отладку по USB
3. Подключите телефон к компьютеру
4. Выполните: `flutter run`
